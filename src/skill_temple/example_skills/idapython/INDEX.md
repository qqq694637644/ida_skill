# IDAPython Example Skill Index

This packaged example demonstrates the Skill Temple API shape. Replace or augment it with your full skill set in a deployment-specific skills directory.

- `SKILL.md`: activation rules, module router, critical rules, and anti-patterns.
- `docs/idautils.md`: iteration helpers such as `Functions`, `FuncItems`, `XrefsTo`, and `Strings`.
- `docs/ida_hexrays.md`: Hex-Rays decompiler basics, `decompile`, and ctree visitor patterns.
